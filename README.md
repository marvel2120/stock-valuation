# A股股票估值分析工具

基于多模型（DCF / PE-PB / Graham / PEG-ROE）的 A 股内在价值评估 Web 工具。

## 快速开始

```bash
cd backend
python3 main.py
# 打开浏览器访问 http://localhost:8000
```

## 功能

- 📊 **DCF 现金流折现**：基于自由现金流预测的绝对估值
- 📈 **PE/PB 相对估值**：基于合理市盈率和市净率的相对估值
- 🔢 **Graham Number**：格雷厄姆经典估值公式
- 🎯 **PEG/ROE 多维估值**：基于增长率和 ROE 的综合评分
- 🔍 支持 A 股代码/名称模糊搜索
- 📋 显示关键财务指标（EPS、ROE、ROA、毛利率等）

## 技术栈

- **后端**：Python 3 + 内置 `http.server`（零额外依赖）
- **数据源**：AKShare
- **前端**：纯 HTML/CSS/JS（无需 npm/构建工具）

## 使用方法

1. 在搜索框中输入 A 股代码或名称（如：600519、贵州茅台）
2. 点击"查询估值"
3. 查看顶部仪表盘（当前股价、综合内在价值、安全边际）
4. 切换 Tab 查看各模型详细分析
5. 底部查看关键财务数据

## 估值模型权重

| 模型 | 权重 |
|------|------|
| DCF 现金流折现 | 40% |
| PE/PB 相对估值 | 30% |
| Graham Number | 15% |
| PEG/ROE 估值 | 15% |
